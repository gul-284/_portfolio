Computer Science Student | Aspiring AI Engineer

A beautiful, high-performance, **Sky Blue** themed portfolio built from scratch with HTML, CSS, and JavaScript.

## 📁 How to Customize:
- **Change Links:** Open `index.html` and look for the `href` attributes in the Projects and Contact sections.
- **Add Images:** Create an `images` folder in your repo, upload your photos, and link them using `<img src="images/yourphoto.jpg">`.
- **Certifications:** Update the `Certifications` section as you earn new badges!
